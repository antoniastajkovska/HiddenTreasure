# Hidden Treasure

## Controls  
- **Up Arrow** → Move forward  
- **Down Arrow** → Move back  
- **Left Arrow** → Move left  
- **Right Arrow** → Move right  
- **Space** → Jump  
- **A / D** → Move the camera left or right  

## Goal  
Collect all the coins and find the hidden treasure before the timer expires!
